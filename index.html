CSS Fundamentals

CSS Statements

Four main components include the following:

  1. Declaration:the statement as a whole with curly braces.
  2. Selectorthe name of the part you want to style.
  3. Propertyproperties to edit(e.g., width, height, background, etc.)
  4. Value:numbers, colors, urls - things to fill in for the property.

Classes - Remember to use .

  • Classes start with . in the selector .git
  • They can safely be called on multiple elements.
  • Very useful, but things can get complicated quickly.

ids - Use the #

  • start with a pound
  • should only be called on one element #hello
  • Cool feature - Run your app and add #cool to the url.
